\begin{itemize}
\item Data: mobile phone-based tracking data provided by Teralytix
\item Residualize journeys between origin-destination pair ($ij$) \\ 
$\text{journeys}_{ijt} = \alpha + \vartheta_{ij} + \gamma_t + \text{weather}_{ijt} + \text{holiday}_{ijt} + \varepsilon_{ijt} $
\item $\gamma_t$ contains dow, woy, month
\item interactions between $\vartheta_{ij} \times \gamma_t$: no interaction, interaction w/ woy and month, and fully interacted
\item classification in the maps uses Fisher-Jenks: minimize variance within clusters, while maximizing variance between the classifications
\end{itemize}
